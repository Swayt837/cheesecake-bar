# Cheesecake Bar - Site Vitrine

## Project Overview
Site vitrine moderne pour un bar à cheesecakes, développé avec React, TypeScript et Vite.

## Tech Stack
- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: CSS Modules / Tailwind (à configurer)
- **Testing**: Vitest + React Testing Library

## Project Structure
```
src/
├── components/
│   ├── layout/        # Header, Footer, Navigation
│   ├── sections/      # Hero, Menu, About, Contact, Gallery
│   └── ui/            # Buttons, Cards, Modal, etc.
├── pages/             # Page components
├── assets/
│   ├── images/        # Photos, backgrounds
│   └── icons/         # SVG icons
├── styles/            # Global styles, variables
├── data/              # Menu items, testimonials
├── hooks/             # Custom React hooks
├── types/             # TypeScript types
└── utils/             # Helper functions
```


## Design Guidelines
- **Colors**: Warm, appetizing colors (cream, brown, berry)
- **Typography**: Elegant, readable fonts
- **Images**: High-quality food photography
- **Responsive**: Mobile-first design

## Key Pages/Sections
1. **Hero**: Eye-catching intro with CTA
2. **Menu**: Showcase cheesecakes with prices
3. **About**: Story of the bar
4. **Gallery**: Photo gallery
5. **Contact**: Location, hours, contact form
6. **Testimonials**: Customer reviews
