# BMAD Help

Welcome to BMAD (Breakthrough Method of Agile AI Driven Development) for the Cheesecake Bar project!

## Available Agents

| Command | Agent | Description |
|---------|-------|-------------|
| `/pm` | John (Product Manager) | PRD creation, requirements, user stories |
| `/dev` | Amelia (Developer) | Implementation, coding, tests |
| `/architect` | Architect | System design, tech decisions |
| `/qa` | QA Engineer | Testing, quality assurance |

## Quick Start Workflows

### For a New Feature:
1. `/pm` → Create PRD with user requirements
2. `/architect` → Design the technical approach
3. `/dev` → Implement the feature
4. `/qa` → Test and validate

### For Bug Fixes:
1. `/qa` → Identify and document the bug
2. `/dev` → Fix and test
3. `/qa` → Validate the fix
